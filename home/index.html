Fast, Reliable, Cheap.

This PC Offers a budget that is afforadable but can also reliable and can run many of the latest games today.

The CPU & GPU

The Ryzen 7 3700X CPU is powerful enough to run most modern games, apps, and much more to get what you need done! Along with a Geforce GTX 1650 Graphics Card that is not overpriced but can still handle most modern games.

The Memory

This PC has DDR4 16GB of Memory with speeds up to 3200MHz, which is enough memory and speed for gaming and much more tasks that you may need to complete. 16GB is the standard for most gaming/workstation Computers.

The Storage

The storage in this PC has 500GB SSD, this is the perfect amount of storage because of size and price. This option can be upgraded later if you need more storage then 500GB, however 500GB is the standard for home computers.

The Motherboard

The Tomahawk Max II is a very powerful motherboard that is able to support the cpu of the Ryzen 7 3700X CPU along with the Geforce 1650 Graphics card, along with many of our other accerries that we have connected this PC.

The Power Supply

The power supply we have chose for this computer is the RM650X from corsair, this allows us to have 650 watts of power to be able to power the CPU, GPU, Motherboard, and way more on our computer.
